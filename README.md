# Liberty NFT Market 🎨

## Overview
Liberty NFT Market is a sleek, professional static website template designed for NFT marketplaces. It allows users to **create, sell, and collect top NFTs** in a modern, responsive interface. Perfect for showcasing digital art, blockchain collectibles, music NFTs, and more.

This is a fully static landing page built with clean, semantic HTML and powered by Bootstrap for quick deployment.

![Hero Banner](./assets/img/banner-01.png)

## Key Features
- **Responsive Navigation**: Hamburger menu with offcanvas sidebar (mobile-friendly), search bar, links to Home, Explore, Item Details, Author, Create Yours.
- **Hero Section**: Eye-catching banner with project title, description, and CTAs (Explore NFTs, Watch Videos).
- **Categories**: Grid of 6 NFT categories (Blockchain, Digital Art, Music Art, Virtual World, Valuable, Triple NFT) with icons.
- **Hot Collections**: Showcase of featured collections (e.g., Bored Ape, Genesis) with item counts, categories, and Explore buttons.
- **Create NFT Guide**: 3-step process (Set up wallet, Add digital NFT, Sell for profit) with icons and descriptions.
- **Marketplace Items**: Live auction cards showing NFTs with current bids (ETH/USD), countdown timers, author profiles, and View Details links.
- **Footer**: Copyright and author credit (BY SAHAR JW).

## Technologies Used
| Category | Technologies |
|----------|--------------|
| **Structure** | HTML5 |
| **Styling** | Bootstrap 5.3.8 (CSS), Custom CSS (`assets/css/style.css`), Google Fonts (Karla, Nunito) |
| **JavaScript** | Bootstrap JS bundle (for offcanvas, responsive components) |
| **Assets** | Static images (PNG/JPG) for logos, banners, icons, NFT previews, authors |
| **Other** | Semantic HTML, CSS variables (e.g., `--primary-color`), responsive design |

No backend or database – purely frontend for demo/portfolio use.

## How to Run Locally
1. Clone/open the repo folder.
2. Open `index.html` in any modern browser (Chrome, Firefox, etc.).
3. **Command to preview**: `start index.html` (opens in default browser).
